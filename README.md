## Descripción
Este repositorio contiene una colección de proyectos y ejercicios desarrollados durante el curso de **Tópicos Avanzados de Programación**. La organización sigue un enfoque didáctico para facilitar la consulta de conceptos clave en Java, desde interfaces gráficas hasta persistencia de datos y multihilos.
